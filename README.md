# RAG Chatbot

## Overview
A Retrieval-Augmented Generation (RAG) chatbot built using Python, LangChain, Hugging Face, FAISS, and Streamlit. The system retrieves relevant information from documents and generates context-aware responses using Large Language Models.

## Features
- Document ingestion and processing
- Semantic search using vector embeddings
- Context-aware response generation
- Streamlit-based interactive interface
- Experiment tracking using MLflow

## Tech Stack
- Python
- LangChain
- Hugging Face Transformers
- FAISS
- Streamlit
- MLflow

## Installation

pip install -r requirements.txt

streamlit run app.py
