# RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using Python, LangChain, Hugging Face, and FAISS.

## Features

- Document Retrieval
- Semantic Search
- Context-Aware Responses
- LLM Integration
- Streamlit Interface

## Tech Stack

- Python
- LangChain
- Hugging Face
- FAISS
- Streamlit
