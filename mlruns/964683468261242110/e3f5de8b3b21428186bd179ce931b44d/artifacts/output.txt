Supervised learning is the machine learning task of learning a function that maps an input to an output based on example input-output pairs. It is a type of learning where a training set of examples with the correct responses (targets) is provided, and the algorithm generalises to respond correctly to all possible inputs. In supervised learning, each example in the training set is a pair consisting of an input object and an output value, and the goal is to train the model so that it can predict the output when it is given new data.
