Intelligent agents are composed of several key aspects:

1. **Agents and Environments**: This refers to the interaction between an agent and its environment. An example given is the vacuum cleaner mini-world.

2. **Rationality**: This concept is crucial for intelligent agents. It is related to omniscience (having complete knowledge) and autonomy (the ability to act independently).

3. **The Nature of Environment and Specifying the Task Environment**: The environment in which an agent operates can have various properties. Understanding these properties is essential for specifying the task environment.

4. **Properties of Task Environments**: Task environments have distinct characteristics that influence how an agent operates.

5. **Examples of Task Environments**: Various examples illustrate the diversity of environments in which intelligent agents can function.

6. **The Structure of Agents**: Agents can be structured in different ways, including:
   - **Simple Reflex Agents**: These agents react to the current state of the environment.
   - **Model-Based Reflex Agents**: These agents maintain an internal model of the environment to make decisions.
   - **Model-Based, Goal-Based Agents**: These agents have specific goals and use their internal models to achieve them.
   - **Utility-Based Agents**: These agents make decisions based on a utility function that determines the desirability of each action.
   - **Learning Agents**: These agents can improve their performance over time through learning from their experiences.

These components and structures contribute to the detailed understanding and functionality of intelligent agents.
